4. Icon Accordion — Rotating Chevron
The chevron rotates 180° when the item opens — a clear directional signal managed by the .open class toggled on .accordion-item. The CSS transform: rotate(180deg) transition is smooth and GPU-accelerated.
All transitions use opacity, transform, and max-height — properties that can be composited on the GPU. Avoid animating height, top, or margin as these trigger layout recalculations.
Add aria-expanded="true/false" to the button and aria-controls pointing to the panel's id. Update these attributes in the JavaScript click handler alongside the class toggle.
Accordions are inherently mobile-friendly — vertical stacking replaces horizontal tabs on narrow viewports. Consider pairing accordions with tab components using a media-query breakpoint swap.

        <span class="btn-icon"><svg ...></svg></span>
        Section Title
      </span>
      <svg class="icon-chevron">...chevron...</svg>
    </button>
    <div class="accordion-body">
