7. スクロール縮小ナビバー
ページを下にスクロールするとナビが高さを縮小する。下のボタンでシミュレーション。
MyApp
  • ホーム
  • 機能
window.addEventListener('scroll', () => {
  const nav = document.querySelector('nav');
  nav.classList.toggle('shrunk', window.scrollY > 50);
});

/* CSS */
nav { height: 56px; transition: height 0.2s ease; }
nav.shrunk { height: 44px; }
