2. Collapsible Sidebar
Click the arrow button inside the sidebar to collapse/expand it.
Dashboard Users Projects Reports Settings

Dashboard

The sidebar collapses to icon-only mode to give more room to content.

/* CSS */
.demo-sidebar.collapsed { width: 56px; }
.demo-sidebar.collapsed .sidebar-label { display: none; }
.demo-sidebar.collapsed .demo-sidebar-link {
  justify-content: center; padding: 0.45rem;
}

/* JS */
document.getElementById('toggle-btn').addEventListener('click', function() {
  document.getElementById('sidebar').classList.toggle('collapsed');
});

5. Mini Sidebar (Icon Only)
Hover over icons to reveal tooltip labels.

Compact Mode

Icon-only sidebar maximizes content space. CSS ::after pseudo-element creates tooltips — no JavaScript needed.

/* ツールチップ: ::after 疑似要素で実装 (JS不要) */
.mini-sidebar-btn::after {
  content: attr(data-tooltip); /* data-tooltip 属性の値を取得 */
  position: absolute;
  left: calc(100% + 8px);
  top: 50%;
  transform: translateY(-50%);
  background: var(--gray-900);
  color: #fff;
  padding: 0.3rem 0.65rem;
  border-radius: var(--radius);
  font-size: 0.78rem;
  white-space: nowrap;
  opacity: 0;
  pointer-events: none;
  transition: opacity var(--transition);
}
.mini-sidebar-btn:hover::after { opacity: 1; }

<!-- HTML -->
<button class="mini-sidebar-btn" data-tooltip="Dashboard">
  <svg ...></svg>
</button>

Click the button to open the right drawer panel.

/* 右ドロワー: transform で画面外から */
.demo-right-drawer {
  position: absolute;
  top: 0; right: 0;
  width: 260px; height: 100%;
  transform: translateX(100%); /* 初期: 画面外(右) */
  transition: transform var(--transition-slow);
}
.demo-right-drawer.open {
  transform: translateX(0); /* スライドイン */
}

/* JS: openクラスを付け外しするだけ */
openBtn.addEventListener('click', function() {
  drawer.classList.add('open');
  overlay.classList.add('open');
});
closeBtn.addEventListener('click', function() {
  drawer.classList.remove('open');
  overlay.classList.remove('open');
});
