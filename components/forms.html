Forms

テキスト入力・セレクト・ファイルアップロードなど、ユーザー入力のためのコンポーネント集。状態（通常/フォーカス/エラー/成功/無効）を色で明確に区別する。

1. テキストインプットの状態
5つの状態を視覚的に区別する。エラーはdanger色、成功はsuccess色のみ使用。
有効なメールアドレスを入力してください
利用可能なメールアドレスです
このフィールドは変更できません
<!-- 通常 -->
<input type="text" class="form-input" placeholder="入力">

<!-- エラー -->
<input type="text" class="form-input error">
<div class="form-helper error">エラーメッセージ</div>

<!-- 成功 -->
<input type="text" class="form-input success">
<div class="form-helper success">成功メッセージ</div>

<!-- 無効 -->
<input type="text" class="form-input" disabled>
2. フローティングラベル
フォーカス時や入力済み時にラベルが上に移動。省スペースかつフォーカス中もラベルが見える。 
/* CSS */
.float-group { position: relative; }
.float-input { padding: 1.4rem 0.75rem 0.4rem; }
.float-label {
  position: absolute; left: 0.75rem; top: 50%;
  transform: translateY(-50%);
  transition: all 0.15s ease;
  pointer-events: none;
}
.float-input:focus ~ .float-label,
.float-input:not(:placeholder-shown) ~ .float-label {
  top: 0.5rem; transform: translateY(0) scale(0.78);
  color: var(--primary);
}

<!-- HTML: placeholder=" " (空白) が必須 -->
<div class="float-group">
  <input type="text" class="float-input" placeholder=" ">
  <label class="float-label">ラベル</label>
</div>

6. テキストエリア ＋ 文字数カウンター
0 / 200
<textarea class="form-textarea" id="comment" rows="4"
  maxlength="200" placeholder="..."></textarea>
<div class="char-counter" id="char-count">0 / 200</div>

<script>
document.getElementById('comment').addEventListener('input', function() {
  const len = this.value.length;
  const el  = document.getElementById('char-count');
  el.textContent = len + ' / 200';
  el.className = 'char-counter' +
    (len > 180 ? ' near-limit' : '') +
    (len >= 200 ? ' over-limit' : '');
});
</script>
