8 avatar variants — built with pure CSS gradients, no real images required. Copy-paste ready.

1. Circle Avatar Gradient Placeholder

Sizes: 32px · 48px · 64px · 80px. Pure CSS gradient — no image required.

<style>

  width: 48px; height: 48px;
  border-radius: 50%;
  display: inline-flex; align-items: center; justify-content: center;
  overflow: hidden; color: white; font-weight: 600; font-size: 0.9rem;
}
.avatar-sm { width: 32px; height: 32px; font-size: 0.72rem; }
.avatar-lg { width: 64px; height: 64px; font-size: 1.1rem; }
.avatar-xl { width: 80px; height: 80px; font-size: 1.3rem; }
.avatar-gradient-1 { background: linear-gradient(135deg, #6366f1, #8b5cf6); }
.avatar-gradient-2 { background: linear-gradient(135deg, #10b981, #3b82f6); }
.avatar-gradient-3 { background: linear-gradient(135deg, #f59e0b, #ef4444); }
.avatar-gradient-4 { background: linear-gradient(135deg, #ec4899, #8b5cf6); }
</style>

4. Avatar Group Overlap → Hover to Spread

Hover over the group to spread the avatars apart.

+3
+5
<!-- Avatar Group: overlapping with hover-spread -->
<style>
.avatar-group { display: flex; align-items: center; }
.avatar-group .avatar {
  margin-left: -14px;
  border: 2px solid white;
  transition: margin-left 0.3s cubic-bezier(.4,0,.2,1), transform 0.3s;
  cursor: pointer;
}
.avatar-group .avatar:first-child { margin-left: 0; }
.avatar-group:hover .avatar        { margin-left: 4px; }
.avatar-group .avatar:hover        { transform: translateY(-4px) scale(1.08); z-index: 10; }
.avatar-group-count {
  margin-left: -14px;
  width: 48px; height: 48px; border-radius: 50%;
  background: #f1f5f9; border: 2px solid white;
  display: inline-flex; align-items: center; justify-content: center;
  font-size: 0.75rem; font-weight: 700; color: #64748b;
  transition: margin-left 0.3s cubic-bezier(.4,0,.2,1);
}
.avatar-group:hover .avatar-group-count { margin-left: 4px; }
</style>
