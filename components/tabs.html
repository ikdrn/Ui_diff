Tabs

8 tab navigation variants — all powered by Vanilla JS. Click any tab to switch panels.

1. Basic Tabs

Overview

Simple horizontal tabs with a bottom border indicator. The active tab receives a colored underline that matches the primary brand color. Clean and minimal — suitable for most use cases.

Features

Lightweight implementation with no dependencies. Keyboard-accessible tab list using role="tablist" and role="tab" attributes for screen-reader compatibility.

Pricing

Free to use. Copy the HTML/CSS/JS snippet directly into your project. No attribution required.

Support

Community-supported via GitHub. Check the issue tracker for known bugs and feature requests.

Design Phase

The sliding underline transitions smoothly using CSS left and width properties. JavaScript measures each tab button's offset and width on click to reposition the indicator with a cubic-bezier easing.

7. Animated Content Tabs

Introduction

Tab content fades in with a slight upward translateY motion — a polished micro-animation that adds perceived quality. The transition uses opacity and transform for GPU-accelerated performance. No layout recalculation triggered.

Installation

No package manager needed. Copy the CSS class definitions and the initTabs() function into your project. Works in all modern browsers without polyfills.

Basic Usage

Add .tabs-container and the variant class to a wrapper element. Place .tab-btn elements inside a .tab-list and corresponding .tab-panel elements as siblings or inside a .tab-panels wrapper.

Advanced Configuration

Override CSS custom properties to retheme any variant. The initTabs() function accepts any container element, so you can initialize dynamically injected tab groups at runtime.

<!-- Content animates via opacity + translateY CSS transition -->
<!-- .tab-panel default: opacity:0; transform:translateY(8px); height:0 -->
<!-- .tab-panel.active:  opacity:1; transform:translateY(0);   height:auto -->

  </div>

    <div class="tab-panel active">...</div>
    <div class="tab-panel">...</div>
  </div>
</div>
