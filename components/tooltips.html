Tooltips

CSS-only tooltips using :hover and position: absolute. No JavaScript required — just hover any element to reveal its tooltip.

💡

Pure CSS implementation. All tooltip variants below use only .tooltip-wrap + .tooltip-box with directional modifier classes. Arrows are rendered via the CSS ::after border trick. The opacity + pointer-events: none pattern prevents the tooltip from stealing mouse events while hidden.

1. Top Tooltip
Tooltip appears above
I'm above you!
Helpful info here
<!-- Top tooltip -->
<div class="tooltip-wrap tooltip-top">
  <button>Hover me</button>
  <div class="tooltip-box">Tooltip appears above</div>
</div>

<style>
.tooltip-wrap { position: relative; display: inline-block; }
.tooltip-wrap .tooltip-box {
  position: absolute;
  background: #1e293b; color: #f1f5f9;
  padding: 0.4rem 0.8rem; border-radius: 6px;
  font-size: 0.8rem; white-space: nowrap;
  opacity: 0; pointer-events: none;
  transition: opacity 0.2s, transform 0.2s;
  z-index: 9999;
}
.tooltip-wrap:hover .tooltip-box { opacity: 1; }
.tooltip-top .tooltip-box {
  bottom: calc(100% + 8px);
  left: 50%; transform: translateX(-50%) translateY(4px);
}
.tooltip-top:hover .tooltip-box { transform: translateX(-50%) translateY(0); }
</style>

</div>

<style>
/* Position the box above */
.tooltip-arrow .tooltip-box {
  bottom: calc(100% + 10px);
  left: 50%; transform: translateX(-50%) translateY(4px);
}
.tooltip-arrow:hover .tooltip-box { transform: translateX(-50%) translateY(0); }

/* Downward-pointing arrow via border trick */
.tooltip-arrow .tooltip-box::after {
  content: '';
  position: absolute;
  top: 100%; left: 50%; transform: translateX(-50%);
  border-width: 6px; border-style: solid;
  border-color: #1e293b transparent transparent transparent;
}
</style>
