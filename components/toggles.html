Toggles

8 toggle & switch variants — all built with checkbox input + custom CSS. No JavaScript required for the toggle logic.

1. iOS Style Toggle iOS / Classic
Checked (green)
Unchecked (gray)
Disabled 
<!-- iOS Style Toggle -->
<style>
.toggle-ios .toggle-track {
  width: 51px; height: 31px;
  background: #cbd5e1;
  border-radius: 9999px;
  transition: background 0.35s cubic-bezier(.4,0,.2,1);
}
.toggle-ios input:checked + .toggle-track { background: #34c759; }
.toggle-ios .toggle-thumb {
  top: 2px; left: 2px;
  width: 27px; height: 27px;
  box-shadow: 0 2px 6px rgba(0,0,0,0.25);
  transition: transform 0.35s cubic-bezier(.4,0,.2,1);
}
.toggle-ios input:checked ~ .toggle-thumb { transform: translateX(20px); }
</style>

<label class="toggle-wrap toggle-ios">
  <input type="checkbox">
  <span class="toggle-track"></span>
  <span class="toggle-thumb"></span>
</label>

8. Animated Icon Toggle X → Checkmark Morph
Checked — shows checkmark
Unchecked — shows X

SVG icons fade & scale between states using CSS transitions. No JS needed.

<!-- Animated SVG Toggle: X morphs to checkmark -->
<style>
.toggle-animated .toggle-track {
  background: #cbd5e1; transition: background 0.3s;
}
.toggle-animated input:checked + .toggle-track { background: #10b981; }
.toggle-animated .toggle-thumb {
  display: flex; align-items: center; justify-content: center;
  transition: transform 0.3s cubic-bezier(.4,0,.2,1);
}
.toggle-animated input:checked ~ .toggle-thumb { transform: translateX(28px); }
.icon-x, .icon-check {
  position: absolute; width: 12px; height: 12px;
  transition: opacity 0.2s, transform 0.3s;
}
.icon-x     { opacity: 1; transform: scale(1) rotate(0deg); }
.icon-check { opacity: 0; transform: scale(0.5) rotate(-45deg); }
.toggle-animated input:checked ~ .toggle-thumb .icon-x {
  opacity: 0; transform: scale(0.5) rotate(45deg);
}
.toggle-animated input:checked ~ .toggle-thumb .icon-check {
  opacity: 1; transform: scale(1) rotate(0deg);
}
</style>

<label class="toggle-wrap toggle-animated">
  <input type="checkbox">
  <span class="toggle-track"></span>
  <span class="toggle-thumb">
    <svg class="icon-x" viewBox="0 0 12 12" fill="none" stroke="#ef4444" stroke-width="2.5" stroke-linecap="round">
      <line x1="2" y1="2" x2="10" y2="10"/><line x1="10" y1="2" x2="2" y2="10"/>
    </svg>
    <svg class="icon-check" viewBox="0 0 12 12" fill="none" stroke="#10b981" stroke-width="2.5" stroke-linecap="round" stroke-linejoin="round">
      <polyline points="1.5,6 4.5,9.5 10.5,2.5"/>
    </svg>
  </span>
</label>
